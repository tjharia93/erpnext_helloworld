# ERPNext HelloWorld App

A minimal custom app for ERPNext v15 with a single **Hello World** script report.

## Report Included
- Hello World (Script Report under Sales Invoice)

## Installation
1. Install from GitHub in Frappe Cloud:
   - Apps → Install from GitHub
   - Repo URL: https://github.com/<your-username>/erpnext_helloworld
   - Branch: main

2. Search "Hello World" under **Reports** in ERPNext.
